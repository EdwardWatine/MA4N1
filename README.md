# MA4N1 Project

Proving Cauchy's Theorem on a triangle using Gorsat's method

## Installation

Ensure that you have installed `lean4` and `lake`.
Clone the git repo.

Run 
```bash
lake update
lake exe cache get
```

And you are good to go!
